⬇️ Fazer Backup
Baixa um arquivo JSON com todos os seus dados. Salve esse arquivo em lugar seguro — ele serve para restaurar tudo em caso de problema.
🔐 Proteção com senha SEM SENHA
Opcional. Se preencher, o arquivo ficará criptografado e só abrirá com essa senha. Se deixar vazio, o backup é salvo sem proteção.
📅
⬆️ Restaurar Backup
Importa um arquivo de backup salvo anteriormente. Os dados atuais serão substituídos pelo conteúdo do arquivo. Se o backup tiver senha, você será solicitado a digitá-la.
🚨 ATENÇÃO: Esta ação substitui os dados atuais pelos dados do backup. Certifique-se de que está importando o arquivo correto.
